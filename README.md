# Kişisel Web Sayfası (Web CBS - Ödev 1)

Bu klasör, 3 farklı HTML sayfası içerir:
- index.html (Hakkımda)
- hobilerim.html (Leaflet harita + gezdiğim yerler)
- iletisim.html (İletişim)

## GitHub Pages ile yayınlama (kısa)
1. Bu klasörü bir GitHub reposuna yükle
2. Repo > Settings > Pages
3. Source: Deploy from a branch, Branch: main / root
4. Adres: https://<kullaniciadi>.github.io/<repo-adi>/

> Not: Harita için Leaflet CDN kullanılmıştır.
